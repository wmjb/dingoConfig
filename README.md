# dingoConfig

dingoConfig is a .NET application with a Blazor Server web UI for managing `dingoPDM`, `dingoPDM-Max`, `CANBoard`, etc. through various communication adapters (`USB`, `SLCAN`, `PCAN`, `Simulated`).
